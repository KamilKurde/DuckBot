# DuckBot ⚡

> An educational logic-puzzle game about electricity — route voltage from generators, through logic gates, to their correct end-points. Built in Unity by a 3-person team, with a stylized, colorful look inspired by Overcooked.
![DuckBot screenshot](screenshots/duckbot.png)

▶️ **[Download & play](https://github.com/KamilKurde/DuckBot/releases)** — see the Releases page.

## About
DuckBot is a logic puzzle game with an educational twist: it shows how electrical **voltage** behaves through hands-on puzzles. Using tools like **logic gates**, the player guides voltage from generators to their correct **end-points**, solving increasingly elaborate circuits. The goal is to clear every level by completing its circuit. Visually, the game aims for a stylized, friendly 3D look in the spirit of **Overcooked**.

## Features
- ⚡ **Voltage-based logic puzzles** — connect generators to their correct end-points
- 🔀 **Logic gates & electrical tools** as the core puzzle mechanics
- 🎓 **Educational angle** — teaches how voltage and circuits work
- 🧩 **Hand-designed levels** with rising complexity
- 🎨 **Custom 3D models, textures and materials** with a stylized Overcooked-like art style
- ✨ **Particle effects, audio and a custom UI**
